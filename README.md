# Project 7 - PHP E-commerce Platform

This project is a PHP-based e-commerce application developed as part of a PHP course.

## Features
- **Modern UI**: A responsive and stylish interface for products and orders.
- **Admin Panel**: Manage users and orders effectively.
- **Cart System**: Easy-to-use cart and checkout functionality.

## Setup
1. Clone the repository into your XAMPP `htdocs` folder.
2. Import the database schema from `db_connect.php` comments into your MySQL database named `project`.
3. Update `db_connect.php` with your database credentials.
4. Run the project through `http://localhost/phpcourse/project7`.
